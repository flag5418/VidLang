# VidLang 详细设计 — 首页与导航设计

## 一、底部导航

```
┌─────────────────────────────────────────────────┐
│                                                   │
│    Learn       Words       Profile                │
│                                                   │
└─────────────────────────────────────────────────┘
```

注意：**没有单独的 Discover tab**。原因：
- 作为个人开发/小团队产品，前期重点是"工具属性"，用户自己导入内容学习
- Discover（发现/推荐/社区）适合平台型产品，后续再考虑

## 五、导航流程总结

```
App Launch
    ↓
Login (Supabase Auth)
    ↓
MainPage
├── Learn Tab
│   ├── Tap content card → PlayerPage / ArticleReaderPage / SongPlayerPage
│   ├── Tap folder card → FolderDetailPage
│   ├── Tap import button → import flow
│   └── Stats → (inline)
│
├── Words Tab
│   ├── Tap word → WordDetailPage
│   ├── Tap Review → ReviewSession
│   └── Filter → filtered list
│
└── Profile Tab
    ├── Stats → StatsDetailPage
    ├── Subscription → SubscriptionPage
    └── Settings → SettingsPage
```
