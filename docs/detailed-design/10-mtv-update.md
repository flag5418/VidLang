# VidLang 更新 — MTV 方案修正

## 一、MTV 学习 = 纯正的"跟唱评分"

用户提出的方案完全可行：

> 用户戴耳机听 MTV → 耳机麦克风采到的主要是用户自己的声音（耳机漏音极低）→ 录音上传评分

### 1.1 技术流程

```
用户播放 MTV（正常音量）
    ↓
用户戴耳机跟唱
    ↓
用户点击录音按钮
    ↓
系统开始录音（麦克风录制用户声音）
    ↓ 同时视频正常播放，用户通过耳机听原唱
    ↓
录音停止
    ↓
录音文件 = 用户的跟唱声音（几乎没有原唱混入）
    ↓
上传到声通评分
    ↓
评分维度：发音准确度、流利度、完整度
    ↓
展示逐词评分反馈
```

### 1.2 和视频跟读的关系

```
视频跟读：                 MTV 跟唱：
耳朵听原声 ✓             耳朵听原唱 ✓
嘴巴跟读 ✓               嘴巴跟唱 ✓
麦克风采用户 ✓            麦克风采用户 ✓
声通英文评分 ✓            声通英文评分 ✓
UI: DubPanel              UI: DubPanel（复用）

差异：
MTV 有歌词时间轴
MTV 需要一个"只录音不听视频音频"的录音模式
```

### 1.3 和卡拉OK评分的区别

```
卡拉OK：分离伴奏+人声 → 比对人声音准
VidLang MTV：不分离，用户戴耳机 → 比对发音准确度

两者不矛盾，VidLang 不需要做"音准评分"，只需要做"发音评分"
发音评分对英语学习更有实际价值
```

### 1.4 MTV 播放器的录音模式

关键改动：录音时**不改变视频音量**（不调低），因为用户戴着耳机听，麦克风录不到耳机声音。

```
录音前：视频音量 100%
录音中：视频音量 100%（用户通过耳机听原唱）
录音后：恢复初始状态
录音文件：用户干净的跟唱声音
```

## 二、MTV 功能总结

| 维度 | 方案 |
|------|------|
| **播放** | 和视频共用 PlayerPage |
| **内容** | VideoInfo + Subtitles(type=lyric) |
| **歌词** | 和字幕共用 Subtitles 表，type 区分 |
| **显示** | PlayerPage 切换"字幕模式"/"歌词模式" |
| **跟唱** | 复用 DubPanel，录音不调低音量 |
| **评分** | 声通英文发音评分（无需音乐评分） |
| **查词** | 点击歌词单词 → WordCard |
| **测试** | 复用统一测试引擎，数据源 = 歌词句子 |
