# VidLang - 设计风格指南

## 一、品牌定位

### 1.1 产品简介

**VidLang** 是一款通过视频学习英语的应用程序，面向全年龄段用户，提供专业的辅助工具，让用户能够轻松、高效地通过观看视频学习英语。

### 1.2 核心价值

- **轻松学习**：降低学习门槛，让英语学习变得有趣
- **寓教于乐**：通过视频内容激发学习兴趣
- **专业辅助**：提供字幕、分词等专业学习工具
- **全年龄段**：适合儿童到成人的不同学习需求

### 1.3 设计灵感来源

参考 **多邻国 (Duolingo)** 的设计理念：

- 简洁清晰的界面布局
- 圆润柔和的视觉元素
- 游戏化的交互体验
- 轻松活泼的色彩搭配

## 二、设计风格

### 2.1 整体风格

- **柔性设计**：大量使用圆角、弧形元素，避免生硬的直线条
- **轻松松弛**：柔和的色彩过渡，营造舒适的学习氛围
- **圆润友好**：按钮、卡片、图标均采用大圆角设计
- **流畅动效**：平滑的过渡动画，提升交互体验

### 2.2 色彩方案

| 颜色类型       | 色值      | 用途                                                       |
| -------------- | --------- | ---------------------------------------------------------- |
| **主色调**     | `#4ADE80` | 品牌色、主要按钮、进度条、强调元素（翠绿色系，参考多邻国） |
| **主色调浅色** | `#DCFCE7` | 背景、卡片、次要元素                                       |
| **成功色**     | `#22C55E` | 成功状态、完成标记                                         |
| **警告色**     | `#FBBF24` | 警告提示、未完成状态                                       |
| **错误色**     | `#EF4444` | 错误提示、删除操作                                         |
| **文字主色**   | `#1F2937` | 标题、主要文字                                             |
| **文字次要色** | `#6B7280` | 辅助文字、注释                                             |
| **文字占位色** | `#9CA3AF` | 占位符、禁用状态                                           |
| **背景色**     | `#F9FAFB` | 页面背景                                                   |
| **卡片背景**   | `#FFFFFF` | 卡片、容器背景                                             |
| **分隔线**     | `#E5E7EB` | 分隔线、边框                                               |

### 2.3 字体规范

| 字体类型     | 字号 | 字重     | 用途           |
| ------------ | ---- | -------- | -------------- |
| **标题 1**   | 24sp | Bold     | 页面标题       |
| **标题 2**   | 20sp | SemiBold | 模块标题       |
| **标题 3**   | 16sp | SemiBold | 卡片标题       |
| **正文**     | 14sp | Regular  | 主要内容文字   |
| **辅助文字** | 12sp | Regular  | 说明文字、提示 |
| **小字**     | 10sp | Regular  | 标签、状态文字 |

### 2.4 图标风格

- 使用圆润的线性图标
- 图标颜色与文字颜色保持一致
- 图标大小根据上下文调整：20dp（按钮）、24dp（导航）、32dp（功能入口）

### 2.5 间距规范

| 间距类型     | 尺寸 | 用途                   |
| ------------ | ---- | ---------------------- |
| **页面边距** | 16dp | 页面内容与边缘的距离   |
| **卡片间距** | 12dp | 卡片之间的间距         |
| **元素间距** | 8dp  | 按钮、输入框等元素间距 |
| **内容间距** | 4dp  | 文字、图标等内容间距   |

### 2.6 圆角规范

| 元素类型     | 圆角大小       |
| ------------ | -------------- |
| **卡片**     | 16dp（大圆角） |
| **按钮**     | 12dp           |
| **输入框**   | 12dp           |
| **图标按钮** | 50%（圆形）    |
| **视频封面** | 12dp           |

### 2.7 阴影规范

| 元素类型   | 阴影样式                                           |
| ---------- | -------------------------------------------------- |
| **卡片**   | `0 4px 12px rgba(0,0,0,0.05)`（柔和阴影）          |
| **按钮**   | `0 4px 12px rgba(74, 222, 128, 0.3)`（主色调阴影） |
| **悬浮层** | `0 8px 24px rgba(0,0,0,0.1)`                       |

## 三、组件规范

### 3.1 按钮

| 类型         | 样式                                          | 用途               |
| ------------ | --------------------------------------------- | ------------------ |
| **主要按钮** | 主色调背景 + 白色文字 + 圆角12dp              | 主要操作、确认按钮 |
| **次要按钮** | 白色背景 + 主色调边框 + 主色调文字 + 圆角12dp | 次要操作、取消按钮 |
| **文字按钮** | 透明背景 + 主色调文字                         | 链接、跳转操作     |
| **图标按钮** | 圆形背景 + 图标（多邻国风格按压效果）         | 快捷操作           |

**按钮交互效果（参考多邻国"心动"效果）**：

- 默认状态：有柔和阴影
- 按下状态：阴影消失，按钮轻微下沉（transform: translateY(2px)）
- 过渡动画：transition: all 0.15s ease

### 3.2 卡片

- 白色背景
- 16dp 大圆角
- 柔和阴影（0 4px 12px rgba(0,0,0,0.05)）
- 底部有分隔线（除最后一个）

### 3.3 列表项

- 左侧图标/缩略图（可选）
- 中间文字区域（标题 + 描述）
- 右侧操作按钮/箭头
- 底部分隔线

### 3.4 输入框

- 白色背景
- 12dp 圆角
- 边框颜色：`#E5E7EB`（默认）、`#4ADE80`（聚焦）
- 占位符颜色：`#9CA3AF`

## 四、交互规范

### 4.1 点击反馈

- 按钮点击时有缩放效果（scale: 0.98）和下沉效果
- 列表项点击时有背景高亮
- 长按触发操作菜单

### 4.2 加载状态

- 使用骨架屏或加载动画
- 加载完成后平滑过渡

### 4.3 错误提示

- 使用 Toast 或 Snackbar 显示错误信息
- 输入框错误时显示红色边框和错误提示文字

## 五、页面结构

### 5.1 通用布局

```
┌─────────────────────────────────────┐
│           顶部导航栏                │
│  [返回] [标题] [搜索] [+操作]      │
├─────────────────────────────────────┤
│           页面内容区域              │
│                                     │
│                                     │
├─────────────────────────────────────┤
│           底部导航栏（可选）         │
│  [首页] [学习] [我的]               │
└─────────────────────────────────────┘
```

### 5.2 页面列表

| 页面名称     | 路径       | 功能描述                     |
| ------------ | ---------- | ---------------------------- |
| **首页**     | `/`        | 展示学习进度、推荐内容       |
| **文件管理** | `/files`   | 管理视频文件夹和视频文件     |
| **视频播放** | `/play`    | 视频播放、字幕显示、学习工具 |
| **学习中心** | `/study`   | 学习记录、统计分析           |
| **个人中心** | `/profile` | 用户信息、设置               |
