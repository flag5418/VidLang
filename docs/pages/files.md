# 文件管理页面设计说明

## 一、页面概述

文件管理是 VidLang 的核心功能页面，用于管理用户的视频文件和文件夹。设计风格参考多邻国，采用卡片式设计，大量使用圆角和柔和阴影，营造轻松、松弛的学习氛围。

## 三、状态定义

### 3.1 播放状态枚举

```dart
enum PlayStatus {
  notStarted,     // 未播放：完全没有播放过
  playing,        // 播放中：播放一部分后暂停
  current,        // 正在播放：当前正在播放的视频
  completed,      // 播放完成：全部播放结束
}
```

| 状态         | 定义               | 图标 | 颜色 |
| ------------ | ------------------ | ---- | ---- |
| **未播放**   | 完全没有播放过     | ○    | 灰色 |
| **播放中**   | 播放一部分后暂停   | ◐    | 灰色 |
| **正在播放** | 当前正在播放的视频 | ▶    | 绿色 |
| **播放完成** | 全部播放结束       | ✓    | 绿色 |

### 3.2 字幕状态

| 状态       | 图标 | 颜色 | 可测试      |
| ---------- | ---- | ---- | ----------- |
| **有字幕** | 📝🟢 | 绿色 | ✅ 可测试   |
| **无字幕** | 📝🔴 | 红色 | ❌ 不可测试 |

### 3.3 页面状态枚举

```dart
enum PageState {
  empty,        // 空状态：无文件夹/无视频
  loading,      // 加载中
  normal,      // 正常显示
  error,       // 错误状态
}
```

## 七、交互设计

### 7.1 首页交互

| 操作           | 效果                                   |
| -------------- | -------------------------------------- |
| **点击文件夹** | 进入文件夹详情页                       |
| **长按文件夹** | 显示操作菜单（重命名、删除、更改封面） |
| **点击 +**     | 显示菜单（导入文件、启动WiFi传输）     |

### 7.2 文件夹详情页交互

| 操作                   | 效果                                         |
| ---------------------- | -------------------------------------------- |
| **点击顶部播放按钮**   | 进入视频播放页面                             |
| **点击顶部 ...**       | 显示菜单（重命名、删除、测试）               |
| **点击顶部 +**         | 显示菜单（导入文件、重命名、综合测试）       |
| **点击九宫格任意位置** | 切换当前播放视频                             |
| **点击九宫格 ...**     | 显示菜单（重命名、删除、导入字幕、单元测试） |

### 7.3 业务规则

| 规则         | 说明                                                   |
| ------------ | ------------------------------------------------------ |
| **测试权限** | 有字幕（📝🟢）才能测试，无字幕（📝🔴）不能测试         |
| **导入字幕** | 导入后字幕图标从 📝🔴 变为 📝🟢，菜单增加测试按钮      |
| **切换视频** | 点击九宫格任意位置切换当前播放视频，顶部大卡片同步更新 |
